Scholar Spark –Research Platform
Project Overview
Scholar Spark is a modern, responsive marketing website for academic research platform.  
It showcases the platform’s features, team, pricing, and includes a contact/feedback form.  
This project was developed as part of the **Web & Multimedia Development** assignment (Year 1, Semester 2).
Live Website
[View the live site on GitHub Pages](https://Bida25-108.github.io/Scholar Spark/)
Pages Included
- `index.html` – Home page with hero, features, how‑it‑works, testimonial, CTA, and footer.
- `about.html` – Mission, team members, company story, and timeline.
- `platform.html` – Detailed feature descriptions and pricing plans.
- `contact.html` – Feedback form (with required fields, radio buttons, checkbox) and office information.
- `resources.html` – Blog‑style layout with featured article, recent posts, and newsletter signup.
Technologies Used
- HTML5 – Semantic structure
- CSS3 – Custom styling, Flexbox, Grid, media queries
- JavaScript– Mobile navigation toggle, form validation (on contact page)
- Font Awesome – Icons
- Unsplash / Random User – Placeholder images (no local images required)
  Git & GitHub – Version control and hosting via GitHub Pages
  Features
- Fully responsive design (mobile, tablet, desktop)
- Consistent header & footer across all pages
- Active navigation highlighting
- HTML5 form with validation (name, email, required fields, radio buttons, checkbox)
- Hover effects and smooth transitions
- Clean typography and generous white space (following the wireframe)
